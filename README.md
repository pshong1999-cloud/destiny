# 天机排盘 - 八字命理 · 六爻占卜 · 万年历

基于中国传统命理学的多功能排盘网页应用，支持八字排盘、六爻占卜、万年历查询、历史记录与分享导出。

## 🚀 快速启动

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 浏览器打开 http://localhost:5173

# 打包部署
npm run build
```

## 📋 功能清单

### 🌟 八字排盘（专业级）
- ✅ 四柱八字（年柱、月柱、日柱、时柱）
- ✅ 天干地支五行标注（木绿/火红/土金/金白/水蓝）
- ✅ 藏干及藏干十神标注
- ✅ 纳音五行显示
- ✅ 十神标注（比肩/劫财/食神/伤官/偏财/正财/七杀/正官/偏印/正印）
- ✅ 五行分布图 + 缺五行标注
- ✅ 日主强弱判定（得令/得地/得助评分）
- ✅ 格局判定（8种格局类型）
- ✅ 用神与忌神
- ✅ 8种神煞（天乙贵人/驿马/华盖/羊刃/禄神/文昌/将星/桃花）
- ✅ 大运排列（8-10柱）+ 当前大运标记
- ✅ 流年展示（前后5年）
- ✅ 5维度命理解读（性格/事业/婚姻/健康/人际）

### 🪙 六爻占卜
- ✅ 铜钱起卦法（6爻×3枚铜钱手动掷卦）
- ✅ 时间起卦法（以当前时间自动起卦）
- ✅ 随机掷卦一键生成
- ✅ 64卦卦象表（本卦+变卦）
- ✅ 纳甲配爻（天干地支+五行标注）
- ✅ 六亲标注（父母/兄弟/子孙/妻财/官鬼）
- ✅ 世应位置标注
- ✅ 动爻识别与变卦生成
- ✅ 卦辞解读（64卦完整卦辞+建议）

### 📅 万年历工具
- ✅ 阳历 → 农历转换（含干支纪年/生肖/星期）
- ✅ 农历 → 阳历转换（含闰月支持）
- ✅ 月历视图（每日显示农历日期+节气）
- ✅ 月份前后翻页导航
- ✅ 节气标注（每月2个节气精确日期）
- ✅ 今日高亮标记

### 📤 分享导出
- ✅ 导出排盘结果为PNG图片（html2canvas截图）
- ✅ 复制排盘文字到剪贴板（纯文字版，方便微信/QQ分享）
- ✅ 自动保存排盘历史（localStorage，最多20条）
- ✅ 查看历史排盘记录
- ✅ 删除单条/清空全部历史

## 🛠 技术架构

```
destiny/
├── index.html              # 主入口（含4个页面区域+导航栏）
├── package.json            # npm 配置
├── vite.config.js          # Vite 开发服务器配置
├── src/
│   ├── main.js             # 应用入口 + 页面导航逻辑
│   ├── style.css           # 自定义样式（五行颜色/中国风）
│   ├── core/
│   │   ├── bazi-engine.js  # 八字排盘引擎
│   │   ├── interpreter.js  # 命理解读引擎
│   │   ├── liuyao-engine.js# 六爻占卜引擎
│   │   └── export-utils.js # 导出/历史工具
│   └── ui/
│   │   ├── input-form.js   # 八字输入表单
│   │   ├── chart-render.js # 排盘表格渲染
│   │   ├── analysis-render.js # 五行格局分析
│   │   ├── dayun-render.js  # 大运流年
│   │   ├── reading-render.js # 解读文字
│   │   ├── liuyao-form.js   # 六爻输入表单
│   │   ├── liuyao-result.js # 六爻结果渲染
│   │   ├── calendar-render.js # 万年历渲染
│   │   └── history-render.js # 历史记录渲染
└── README.md
```

## ⚠️ 说明

- 八字排盘基于 `lunar-javascript` 库的天文算法级精确计算
- 六爻占卜基于传统铜钱法/时间法和64卦完整体系
- 万年历节气由 `lunar-javascript` 精确计算
- 所有解读为传统命理学知识，仅供参考与学习，不构成科学验证或人生决策依据
